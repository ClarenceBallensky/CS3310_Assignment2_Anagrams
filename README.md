# CS3310_Assignment2_Anagrams
Finds all anagrams for each word in any given text file. 


Text file must formatted with one string on each line.
Program ignores distinctions between upercase and lowercase numbers. For example, “Elvis” and “lives” should be identified as
anagrams of one another.

Program ignores punctuation, including apostrophes. For example, "b'est" and "be'ts" should be identified as anagrams of one another.

Program ignores accent markers. For example, "cät" and "act" should be identified as anagrams of one another. 

## How to run
* clone this GitHub repository 
* open up the command prompt terminal on your machine
* use the change directory command "cd" and navigate to the CS3310_Assignment2_Anagrams > src > main > java
* compile the program using the "javac" command
* run the program using the "java" command, including the full path to your text file as a command line argument
